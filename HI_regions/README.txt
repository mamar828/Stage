This folder contains all the files connected to HI regions' analysis


CO_data                 .fits data cubes of every CO cloud.
data_cube.py            data cube class useful for HI data cubes' manipulation.
LOOP4_bin2.fits         HI data binned 2x2.
LOOP4_FINAL_GLS.fits    raw HI data.
test.py                 test file.
vlb_N4.fits             rotated data cube of the fourth CO cloud to have speed, longitude and latitude as x, y and z respectively.
vlb.fits                rotated data cube of the HI cloud to have speed, longitude and latitude as x, y and z respectively.
